# Lonestar Academy

## Overview

A custom institutional website built for **Lonestar Academy**, designed to give the institution full control over its own content without needing a developer for every change. Built with Next.js on the frontend and a Node.js/Express backend, with MongoDB for data and Cloudinary for media.

## What Admins Can Do

- **Manage Courses:** Create and edit courses with pricing, duration, descriptions, curriculum, and images — all from the dashboard.
- **Manage Blogs:** Publish and manage blog content directly through the admin panel.
- **Handle Enquiries:** A centralized enquiry management system to track and respond to leads.
- **Review Enrollments:** Manage the entire student enrollment pipeline from one place.

## Student Enrollment Workflow

- **Secure Document Uploads:** Students can upload ID proofs, educational certificates, and other supporting documents securely.
- **Online Payments:** Integrated payment gateway for course enrollment fees.
- **Optimized Storage:** Cloudinary handles media and document storage, keeping uploads fast and reliable.

## Why I built this

The institution needed a way to run its entire online presence — courses, blogs, enquiries, enrollments — without depending on a developer for routine updates. So the core focus was building a CMS flexible enough to hand full control to non-technical admins.

## Tech Stack

- Next.js
- Node.js
- Express.js
- MongoDB
- Cloudinary
- Hostinger VPS

## Deployment

- Backend deployed and configured on **Hostinger VPS**, delivering a scalable, secure, and easy-to-manage platform for the institution.
