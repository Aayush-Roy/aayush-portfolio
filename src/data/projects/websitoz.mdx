## Overview

**Websitoz** was my first company as a Full Stack Developer Intern, where I redesigned their website from the ground up using React and Tailwind CSS. The focus was on creating a stunning, animated user experience that stood out.

## Key Features

- **Smooth Scroll Experience**: Implemented Locomotive Scroll for a fluid, immersive scrolling experience across the site.
- **Rich Animations**: Used GSAP and Framer Motion to bring components to life with polished, eye-catching animations.
- **Redesigned UI**: Rebuilt the site's visual design end-to-end for a more modern, engaging look.

## Tech Stack

- React
- Tailwind CSS
- GSAP
